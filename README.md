## Description

The app is about to track all user location on map while moving. When the user stops for five minutes it ends the trip , the app is working in background.


## Getting Started



### Requirements

* iOS 13.0 and later
* XCode Version 13.1


### Running App
* Download project 
* Open it with XCode
* Run with simulator or attach your iphone
